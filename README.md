# 基于代币激励的3D旅游图像建模
## 项目简介
我们是做一款旅游的预体验产品。用户上传景点照片，获得Token 奖励，系统利用用户上传的图片，通过三维重建技术，合成一个360度全景展示。用户可以通过支付Token获取合成的近乎现实的VR实景，来预先体验景点，以获得先验性的评价。
